Labeled October 1944 sortie folders
===================================

Each subfolder is named for a parsed sortie ID from the October 1944 dataset.
Inside each folder:
- a labeled portrait-oriented copy of the front image
- a labeled portrait-oriented copy of the paired back image when one is listed in the dataset

These are copied from Oct1944/IndiaBurma1944Photos, so the labeled folder stays self-contained even if the raw archive folder is moved later.
See manifest.csv for the full mapping.
